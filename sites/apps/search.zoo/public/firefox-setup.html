You can add Zoo Search as a search engine in Firefox to quickly search across all Zoo sites.

Option 1: Automatic (Recommended)

Firefox should automatically detect Zoo Search as an available search engine when you visit the search page.

  1. Visit Zoo Search
  2. Click on the address bar
  3. Look for the "+" icon at the bottom of the dropdown
  4. Click "Add Zoo Search"

Option 2: Manual Setup

Add Zoo Search with a custom keyword:

  1. Right-click the address bar
  2. Select "Add a Keyword for this Search..."
  3. Enter a keyword like "zoo"
  4. Now you can type "zoo your search query" in the address bar

Search URL Format

http://search.zoo/?q={searchTerms}

Zoo Search now uses server-side rendering with a simple form-based interface. The ?q= parameter is supported for direct search queries.
